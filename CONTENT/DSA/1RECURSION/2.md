# n^p || Binary exponentiation
```
n^k=n^(k/2)*n^(k/2) //if k is even
n^k=n^(k/2)*n^(k/2)*n //if k is odd
```

```cpp
int pow(int n,int k){
  if(k==0){
    return 1;
  }
  int x=pow(n,k/2);
  if(k%2==0){
    return x*x;
  }
  return x*x*n;
}
```

https://leetcode.com/problems/powx-n/description
1. n can be a double
2. n can be an negative integer

```cpp
class Solution {
public:
  double pow(double n,long k){
    if(k==0){
      return 1;
    }
    double x=pow(n,k/2);
    if(k%2==0){
      return x*x;
    }
    return x*x*n;
  }

  double myPow(double n, int kInit){
    long k=kInit;
    double tmpResult=pow(n,abs(k));
    return k>=0?tmpResult:1/tmpResult;
  }
};
```
