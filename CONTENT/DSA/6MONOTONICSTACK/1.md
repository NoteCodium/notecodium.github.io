# NGE

1. The stack be increasing from top to bottom
2. because any time an element which came is greater then the top element of the stack, it is their nge

```cpp
stack<int> st;
st.push(0);
vector<int> nge(n,n);
for(int i=1;i<n;i++){
  while(!st.empty() and v[i]>v[st.top()]){
//as long as it is greater then the top element of stack, it will be nge of that element
    nge[st.top()]=i;
    st.pop();
  }
  st.push(i);



}
```
